NCAA Track Comparisons

All of the following comparisons make use of world athletics scores. These are a ranking system designed by World Athletics to provide a single comparable number for every performance across track and field events. These scores generally scale from 1 to 1400 with a higher score representing a more impressive performance.

Compare Conferences

Check the boxes below to have that conference's average score for all events be shown as a line on the graph. Multiple boxes can be selected to compare conferences across events. The number entered will be used to select the amount of performances accounted for in the average. Press UPDATE to display the graph.

Compare Events

Check the boxes below to have the events average score for all conferences be shown as a line on the graph. Multiple boxes can be selected to compare events across the entire NCAA. The number entered will be used to select the amount of performances accounted for in the average, and the event entered will sort the conferences by that events score in descending order. Press UPDATE to display the graph.
